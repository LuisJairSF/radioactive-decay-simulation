# Radioactive Decay Simulation

This project models radioactive decay using a stochastic process based on repeated dice experiments.

## Objective
To verify the exponential decay model:
N(t) = N0 e^(-λt)

## Methodology
- Discrete time steps
- Constant probability of decay
- Statistical analysis of remaining particles

## Tools
- Python
- Excel
- Statistical analysis

## Results
Observed exponential behavior consistent with theoretical prediction.
